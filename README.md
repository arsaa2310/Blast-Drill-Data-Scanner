Aplikasi untuk mendeteksi dan merekam kegiatan blasting per shift di area pertambangan menggunakan **OpenCV** dan **Streamlit**.

## Fitur
- Membantu mengurangi penggunaan kertas.
- Proses deteksi blok/aktivitas menggunakan image processing.
- Menyimpan output ke file CSV.
- Interface sederhana dengan Streamlit.
